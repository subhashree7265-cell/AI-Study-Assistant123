🤖 AI Study Assistant Using Generative AI & Cloud Computing
📌 Project Overview
AI Study Assistant is a Generative AI-based web application developed as part of my IBM Gen AI & Cloud Computing internship.

The main goal of this project is to provide students with an AI-powered learning companion that helps them understand concepts, get explanations, and improve their learning experience.

🎯 Objectives
Develop an AI-based study support system
Provide quick answers to student queries
Understand the implementation of Generative AI concepts
Learn frontend and backend integration
Explore cloud-based application development
✨ Features
✅ AI Study Assistant
✅ Question and Answer Support
✅ User-friendly Web Interface
✅ Backend API using Flask
✅ Generative AI workflow demonstration
✅ Cloud deployment ready structure
